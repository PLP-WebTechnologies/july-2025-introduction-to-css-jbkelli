üíªCategorizing operating systems

Well basically, operating systems are categorized in 4 different categories:

  1. By Functionality
    • Desktop Operating Systems (Windows, macOS, Linux)
    • Mobile Operating Systems (Android, iOS)
    • Server Operating Systems (Windows Server, Linux Server)
    • Embedded Operating Systems (FreeRTOS, Embedded Linux)
  2. By Usage
    • Single-User Opearting Systems (Windows)
    • Multi-User Operating Systems (Unix, Linux)
  3. By Architecture
    • Monolithic Operating Systems (Linux, Windows)
    • Microkernel Operating Systems (QNX, Symbian)
  4. Special-Purpose Operating Systems
    • Real-Time Operating Systems - RTOS (VxWorks, FreeRTOS)
    • Distributed Operating Systems (Cloud computing)
